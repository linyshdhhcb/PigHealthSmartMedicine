# 生猪饲养管理与疾病防治技术要点

## 一、饲养管理

### 1. 环境控制
- **温湿度管理**：保持适宜温度（仔猪30~32℃，育肥猪18~22℃），湿度控制在60%~70%。
- **通风换气**：确保舍内空气流通，减少氨气、硫化氢等有害气体积聚。
- **密度控制**：保育猪≤3头/m²，育肥猪1~1.2头/m²，避免过度拥挤。

### 2. 饲料与饮水
- **营养均衡**：根据生长阶段调整饲料配方，确保蛋白质、能量、微量元素充足。
- **饮水保障**：提供清洁饮水（15~20℃），饮水器流速≥1.5L/min，高温季节可添加0.1%食盐补充电解质。

### 3. 生物安全
- **消毒制度**：每日1~2次带猪消毒，场区外围每周3次喷洒杀虫剂。
- **人员管理**：严格控制人员、车辆进出，建立消毒通道和登记制度。
- **病死猪处理**：无害化处理，严禁随意丢弃或出售。

### 4. 健康档案与监测
- **建立档案**：记录出生、疫苗接种、病史、饲料配方等信息。
- **智能监测**：使用红外测温、称重系统等进行实时健康评估。

## 二、疾病防治

### 1. 疫苗接种（第一道防线）
| 疾病名称                   | 接种建议                                      |
| -------------------------- | --------------------------------------------- |
| 猪瘟                       | 种猪每年2次，仔猪20~30日龄首免，60~70日龄二免 |
| 猪蓝耳病                   | 根据场区情况选择疫苗，注意免疫时机与剂量      |
| 猪链球菌                   | 3~4周龄首免，间隔3周二免                      |
| 猪丹毒、支原体、大肠杆菌等 | 根据流行情况和疫苗效果选择性接种              |

> ⚠️ 注意：疫苗应结合本场实际情况选择，避免盲目接种。

---

### 2. 常见病防治措施

#### 2.1 呼吸道疾病（如支原体、胸膜肺炎）
- **预防**：加强通风、减少应激、疫苗接种。
- **治疗**：
  - 氟苯尼考（10~20mg/kg，肌注，3~5天）
  - 多西环素（饮水，10~20mg/kg/d，5~7天）
  - 混合感染可用替米考星+多西环素联合用药

#### 2.2 消化道疾病（如黄白痢、腹泻）
- **预防**：母猪产前接种大肠杆菌疫苗，仔猪初乳摄入充足。
- **治疗**：
  - 粘杆菌素注射+庆大霉素灌服
  - 补液盐防止脱水

#### 2.3 寄生虫病（如蛔虫、疥螨）
- **预防**：定期驱虫，保持舍内干燥清洁。
- **治疗**：
  - 阿苯达唑、伊维菌素口服或注射
  - 疥螨可外用多拉菌素或伊维菌素涂抹

#### 2.4 猪链球菌病
- **症状**：高热、关节炎、淋巴结化脓等。
- **防治**：
  - 疫苗接种+阿莫西林/恩诺沙星预防
  - 发病猪隔离治疗，局部热敷或涂药膏

---

### 3. 非洲猪瘟（ASF）
- **现状**：无商业化疫苗，防控依赖生物安全。
- **措施**：
  - 严格消毒、禁止外来车辆人员进入
  - 病死猪立即无害化处理
  - 政府提供扑杀补贴，但部分地区落实不到位

---

## 三、综合防控建议

1. **全进全出**：减少交叉感染风险。
2. **分阶段饲养**：根据日龄、体重分群管理。
3. **减少应激**：避免频繁转群、突然换料、高温运输等。
4. **科学用药**：避免滥用抗生素，注意休药期和药物残留。
5. **建立预警机制**：定期检测抗体水平和病原流行情况。

---

## 四、结语

生猪养殖是一项系统工程，饲养管理与疾病防控必须并重。通过科学管理、规范免疫、合理用药和严格生物安全措施，可有效降低疾病发生率，提升养殖效益。
